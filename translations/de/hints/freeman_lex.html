Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur alle vorherigen Hinweise zusammenführen, eine sortierte allgemeine Liste zurückgeben und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

from collections.abc import Iterable


def string_permutations(s: str) -> Iterable[str]:
    
    if len(s) < 2: return [s]
    all = []
    for i, char in enumerate(s):
        remaining_chars = s[:i] + s[i + 1:]
        for perm in string_permutations(remaining_chars):
            all.append(char + perm)
    
    return sorted(all)

Verwenden Sie das Standardmodul itertools und die Funktion permutations, um alle Varianten zu erzeugen, fügen Sie jede mit der Funktion map zu einem einzigen String zusammen (da jede von ihnen ein Tupel ist) und geben Sie eine sortierte Liste zurück. 

from collections.abc import Iterable
from itertools import permutations


def string_permutations(s: str) -> Iterable[str]:
    
    return sorted(map("".join, permutations(s)))
