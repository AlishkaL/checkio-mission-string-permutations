Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę wpaść na jakiś pomysł 💡! Proszę zastosować podejście rekurencyjne. Iterujemy po podanym ciągu i dla każdego znaku łączymy pozostałe znaki, a dla krótszego ciągu wywołujemy funkcję ponownie. Proszę połączyć każdy zwrócony wariant ze znakiem powyżej i dodać wynik do ogólnej sekwencji, którą później posortujesz i zwrócisz.

Ok, od czego powinienem zacząć?

Po pierwsze, musi Pan zdefiniować przypadek brzegowy, gdy s pozostaje równe 0 lub 1. W takim przypadku proszę po prostu zwrócić s. 

if len(s) < 2: return [s]

Proszę zadeklarować zmienną do przechowywania wszystkich permutacji z początkową wartością pustej listy. 

all = []
Co robić dalej?

Teraz należy wykonać iterację po podanym ciągu znaków, aby uzyskać każdy znak. Proszę użyć pętli for i funkcji enumerate(), aby uzyskać zarówno indeksy, jak i znaki. 

for i, char in enumerate(s):
    ...

Dla każdego znaku proszę utworzyć pozostały ciąg za pomocą krojenia. 

remaining_chars = s[:i] + s[i + 1:]
Nadal potrzebuję podpowiedzi.

Teraz wewnątrz powyższej pętli należy utworzyć permutacje dla tego krótszego ciągu i iterować po każdej z nich. Proszę ponownie użyć pętli for i wywołać tę samą funkcję. 

for perm in string_permutations(remaining_chars):
    ...
Co powinienem teraz zrobić?

Proszę połączyć rozdzielony powyższy znak i każdą z permutacji perm krótszego ciągu. W ten sposób poprzez zagnieżdżoną rekurencję zbudują Państwo wszystkie unikalne warianty. Proszę dodać wynik do ogólnej listy all. 

all.append(char + perm)
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi, zwrócić posortowaną listę ogólną i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

from collections.abc import Iterable


def string_permutations(s: str) -> Iterable[str]:
    
    if len(s) < 2: return [s]
    all = []
    for i, char in enumerate(s):
        remaining_chars = s[:i] + s[i + 1:]
        for perm in string_permutations(remaining_chars):
            all.append(char + perm)
    
    return sorted(all)

Proszę użyć standardowego modułu itertools i funkcji permutations, aby wygenerować wszystkie warianty, połączyć każdy z nich w pojedynczy ciąg (ponieważ każdy z nich jest krotką) za pomocą funkcji map i zwrócić posortowaną listę. 

from collections.abc import Iterable
from itertools import permutations


def string_permutations(s: str) -> Iterable[str]:
    
    return sorted(map("".join, permutations(s)))
