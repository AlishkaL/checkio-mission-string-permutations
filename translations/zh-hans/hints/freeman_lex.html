您只需将之前的所有提示连接起来，返回排序后的总列表，并将代码封装为函数表达式。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

from collections.abc import Iterable


def string_permutations(s: str) -> Iterable[str]:
    
    if len(s) < 2: return [s]
    all = []
    for i, char in enumerate(s):
        remaining_chars = s[:i] + s[i + 1:]
        for perm in string_permutations(remaining_chars):
            all.append(char + perm)
    
    return sorted(all)

使用标准模块itertools 和permutations 函数生成所有变体，使用map 函数将每个变体连接成单个字符串（因为每个变体都是一个元组），然后返回排序后的列表。 

from collections.abc import Iterable
from itertools import permutations


def string_permutations(s: str) -> Iterable[str]:
    
    return sorted(map("".join, permutations(s)))
