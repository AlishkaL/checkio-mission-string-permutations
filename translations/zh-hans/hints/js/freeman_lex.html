将分隔的上述字符和较短字符串的每种排列perm 合并。这样，通过嵌套递归，就可以建立所有独特的变体。将结果添加到总列表all 中。

您只需将之前的所有提示连接起来，返回排序后的总列表，并将代码封装成函数表达式。强烈建议对输入和输出数据使用类型提示。

下面是整个解决方案：

function stringPermutations(s: string): string[] {

    if (s.length <= 1) return [s];
    const all = [];
    for (let i = 0; i < s.length; i++) {
        const char = s[i];
        const remainingChars = s.slice(0, i) + s.slice(i + 1);
        for (const perm of stringPermutations(remainingChars)) {
            all.push(char + perm);
        }
    }
    return all.sort();
}
